# Ava 语言设计

## 0. 简介

`Ava` 是一个函数式+面向过程的编程语言，语法特性类似于 `C++`、`Python` 和 `Rust`，你会发现一些以上语言中的特性。

```c++
[]max(int mut x, int mut y) -> int {
    if(x >= y) {
        return x;
    }
    y;
} 

[]swap(int& mut x, int& mut y) {
    int mut t = x;
    x = y;
    y = t;
}

[]main() -> int {
	print("Hello, Ava !\n");
    int mut x = 3;
    int mut y = 4;
    x = 4;
    double a = 3.14;
    swap(x, y);
    int mut m = max(x, y);
    [&]() {
        print(x);
    }();
    0;
}
```
